# Enrolling for a Session

You can explore Public sessions on <PartialExample mentored /> and sign up for sessions in your chosen domain or by a mentor of your choice.

<Admonition type="note">
<ul>
<li>If a session manager invites you to a Public or Private session, you are automatically enrolled to the session.</li>
<li>You cannot enroll for a session after the enrollment limit is reached. If you try to enroll after this limit is reached, a <i>Session seats full</i> message appears.</li>
<li>You can enroll for an ongoing session, which appears as <i>Live</i> on the Home page.</li>
</ul>
</Admonition>

<p>You can enroll for Public sessions in one of the following ways:</p>
<ul>
<li>Using the <b>All sessions</b> tab</li>
<li>Using the mentor profile</li>
<li>Using the Session details page</li>
<li>Using the <b>Search</b> box</li>
</ul>

## Using the All Sessions Tab
<ol>
<li>On the Home page, go to the <b>All sessions</b> tab.</li>
<li>Find a session of interest and click <b>Enroll</b>. The enrolled sessions are listed on the <b>Enrolled sessions</b> tab of the Home page.</li>
<p>
    <img src={PortalEnrollButton} />
</p>
</ol>

## Using the Mentor Profile

<Admonition type="info">
<p>See <a href="mentor-directory">Mentor Directory</a> to learn more about accessing the mentor profile.</p>
</Admonition>
<p><b>To enroll for a session using the mentor profile, do as follows:</b></p>
<ol>
<li>On the Mentor Profile page, go to the <b>Upcoming sessions</b> tab.</li>
<li>Click <b>Enroll</b>. The enrolled sessions are listed on the <b>Enrolled sessions</b> tab of the Home page.</li>
<p>
    <img src={PortalMentorProfileEnroll} />
</p>
</ol>

## Using the Session Details Page

<ol>
<li>Find a session from the <b>All sessions</b> tab or the Mentor Profile page.</li>
<li>Click the session tile to view the Session Details page.</li>
<p>
    <img src={PortalSessionTileEnroll} />
</p>
<li>Click <b>Enroll</b>. The enrolled sessions are listed on the <b>Enrolled sessions</b> tab of the Home page.</li>
<p>
    <img src={PortalSessionDetailsEnroll} />
</p>
</ol>

## Using the Search Box

1. To find upcoming sessions, do any one of the following actions:

    * Enter a search phrase such as the session title in the **Search** box and press **Enter**. The search results appears as session tiles.

    * Search for sessions based on a specific category.

        1. To specify the search category, refer to the following table:

            <table>
            <tr>
                <th>Category</th>
                <th>Actions</th>
            </tr>
            <tr>
                <td>Search Category</td>
                <td>Click the <b>Search</b> box and select a search category such as <b>Mentor name</b>.</td>
            </tr>
            <tr>
                <td>Filter</td>
                <td><ol><li>Click on <b>Filter</b>.</li><li>Select one or more filter checkboxes and click <b>Apply</b>.</li><Admonition type="tip"><p>To clear filters, do one of the following actions:</p><ul><li>Clear the filter checkbox.</li><li>Click <b>Clear all</b> to clear all the applied filters.</li></ul></Admonition></ol></td>
            </tr>
            </table>

        2. Enter the search phrase. Based on the specified search category, the search results display the session tiles.

            ![](media/portal-searchsessions.png)

    <p>
    <Admonition type="tip">
    <p>If there are multiple search results, you can find the respective session using the following ways:</p>
    <ul>
    <li>Viewing the search results on different pages by clicking the <b>Next</b> or <b>Previous</b> buttons.</li>
    <li>Changing the number of search results listed on the page using the <b>Items per page</b> dropdown.</li>
    <li>Applying filters.</li>
    </ul>
    </Admonition>
    </p>

2. Click **Enroll**. The enrolled sessions are listed on the **Enrolled sessions** tab of the Home page.

## Cancelling Your Enrollment

<Admonition type="note">
<ul>
<li>If you were invited to a session, you cannot cancel your enrollment for the session.</li>
<li>You can cancel your enrollment if the mentor has not started the session.</li>
</ul>
</Admonition>

<p><b>To cancel your enrollment, do as follows:</b></p>

<ol>
<li>On your Home page, go to the <b>Enrolled sessions</b> tab.</li>
<li>Click the session tile.</li>
<li>Click <b>Unenroll</b>.</li>
<p>
    <img src={PortalCancelEnrollment} />
</p>
<li>Click <b>Unenroll</b> on the confirmation dialog. The session is removed from the <b>Enrolled sessions</b> tab of the Home page.</li>
</ol>
