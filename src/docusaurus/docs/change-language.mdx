# Changing the Language Settings

You can use the application in a language of your choice.

**To change the language settings, do as follows:**

<Tabs>
<TabItem value="mentored mobile application" label="Mobile App" default>
<ol>
<li>Go to the <b>Application</b> menu <div class="inlineImg"><img src={BurgerMenu}/></div> and select <b>Language</b>.</li>
<li>On the Language page, tap a <b>Language</b> button of your choice.
<br />
<br />
    <img src={LanguageSetting} />
</li>
<li>Tap <b>Submit</b>. The text on the application user interface now appears in the language that you have chosen.</li>
</ol>
</TabItem>
<TabItem value="mentored portal" label="Web Portal">
<ol>
<li>Click the Language drop-down on the top-right corner of the page.
<br />
<br />
    <img src={PortalLanguageSetting} />
</li>
<li>Select a language of your choice. The text on the application user interface now appears in the language that you have chosen.</li>
</ol>
</TabItem>
</Tabs>
