# Setting up and Managing Organizations

<Admonition type="info">    
<p>To learn more about the concept of organizations in <PartialExample mentored />, see <a href="policyarchitecture">Multi-Tenant Architecture</a>.</p>
</Admonition>

You can set up organizations only using the REST APIs of User service and <PartialExample mentored /> service.

Using the REST APIs of User service, you can do the following:

  * Create a System Administrator account.

  * Log in to the System Administrator account.

  * Log in to the Organization Administrator account.

  * Create and manage organizations.

  * Assign an organization administrator.

  * Manage users at an organization.

Using the REST APIs of <PartialExample mentored /> service, organizations can do the following:

* Configure policies.

* Create and manage entity types and entities in the session and profile creation pages.

* Set default rules.
