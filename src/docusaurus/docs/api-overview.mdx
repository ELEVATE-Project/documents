# Overview

To learn more about the concept of organizations, see <a href="/userservice/aboutorganizations" target="_self">About Organizations</a>.

Using the REST APIs of <PartialExample mentored /> service, organizations can do the following:

* Configure policies.

* Create and manage entity types and entities on the session and profile creation pages.

* Set default rules.

## Accessing the API Collection

To access the APIs on Postman, do any one of the following actions:

* Import the JSON and environment file from <a href="https://github.com/ELEVATE-Project/mentoring/tree/master/src/api-doc">https://github.com/ELEVATE-Project/mentoring/tree/master/src/api-doc</a>

* Import the raw URL from <a href="https://raw.githubusercontent.com/ELEVATE-Project/mentoring/master/src/api-doc/MentorED-Mentoring.postman_collection.json">https://raw.githubusercontent.com/ELEVATE-Project/mentoring/master/src/api-doc/MentorED-Mentoring.postman_collection.json</a>
