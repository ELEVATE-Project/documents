# Assigning Sessions to a Mentor

By creating and assigning sessions to a mentor, you can gain an oversight to the mentoring program at an organization and track the progress of sessions.

**To create a session, do as follows:**

<ol>
<li>Go to the <b>Application</b> menu <img src={BurgerMenu}/> and select <b>Workspace</b>.</li>
<li>On the <b>Workspace</b> page, click <b>Manage session</b>.</li>
<li>Click <b>Create session</b>.
<p>
    <img src={PortalSessionDetails} />
</p>
</li>
<li>Optionally, to add a session image, click <b>Add image</b> and do one of the following actions:
    <ul>
    <li>To take a picture, select <b>Take Photo</b>.</li>
    <li>To upload an image, select <b>Choose from library</b>.
        <p>  
        <Admonition type="tip">
        <p>Select an image that is relevant to the session topic.</p>
        </Admonition>
        </p>
    </li>
    </ul>
</li>
<li>Enter the session information in the <b>Session title</b> and <b>Description</b> boxes.</li>
<li>Select the session type, do as follows:
    <ul>
    <li>Click the <b>Session type</b> box.</li>
    <li>Select the session type from the <b>Session type</b> dropdown and click <b>Ok</b>.</li>
    </ul>
</li>
<li>To assign a mentor, do as follows:
    <ul>
    <li>Click <b>+ Add mentor</b>. The Mentor list window appears.</li>
    <li>Find a mentor from the list and click <b>Add</b>.</li>
    </ul>
    <p>  
        <Admonition type="note">
        <p>To change the assigned mentor, click the mentor's name on the Create a new session page and add a new mentor. You cannot change the assigned mentor after publishing the session.</p>
        </Admonition>
    </p>
</li>
<li>To invite mentees, do as follows:
    <ul>
    <li>Click <b>+ Add mentee</b>. The Mentee list window appears.</li>
    <li>Find a mentor from the list and click <b>Add</b>.</li>
    </ul>
    <p>  
        <Admonition type="tip">
        <ul>
        <li>It is optional to invite mentees to a Public session.</li>
        <li>To add another mentee, click <b>+ Add New Mentee</b> and add mentees from the list.</li>
        <li>To remove a mentee, click the mentee's name and click <b>Remove</b> on the Mentee list window.</li>
        </ul>
        </Admonition>
    </p>
</li>
<p>  
    <Admonition type="tip">
    <p>You can find mentees or mentors from the list faster by:</p>
    <ul>
    <li>Searching for the mentor or mentee in the <b>Search</b> box.</li>
    <li>Using filters such as the user's organization or designation.</li>
    <li>Changing the number of users listed on the page using the <b>Items per page</b> dropdown.</li>
    </ul>
    </Admonition>
</p>
<li>To schedule a session, do as follows:
    <ol>
    <li>Click the <b>Start date</b> and <b>End date</b> boxes.</li>
    <li>Select the date and time from the date and time picker.</li>
    <li>Click <img src={SelectDateAndTime} />.</li>
    </ol>
<p>  
    <Admonition type="note">
    <p>The session duration should be at least 30 minutes.</p>
    </Admonition>
</p>
</li>
<li>Select an option from the following boxes:
    <ul>
    <li><b>Recommended for</b></li>
    <li><b>Categories</b></li>
    <li><b>Medium</b></li>
    </ul>
<p>
    <Admonition type="tip">
    <ul>
    <li>To cancel the selection, click the option again.</li>
    <li>To add a new option, do as follows:
        <ol>
        <li>Click <b>+ Add</b> and enter the new option.</li>
        <li>Click <b>Ok</b>.</li>
        </ol>
    </li>
    <li>To select all the options, turn on the toggle.</li>
    </ul>
    </Admonition>
</p>
<p>
    <img src={PortalSessionDetailsOptions} />  
</p> 
</li>
<li>Click <b>Publish and add link</b>. The session is published on the platform and participants can now view your session.
<p>
    <Admonition type="note">
    <p>You must set the meeting platform on the <b>Meeting Link</b> tab.</p>
    </Admonition>
</p>
<p>
    <img src={PortalAddMeetingLink} />  
</p>
<p>
    <Admonition type="tip">
    <p>To go back and edit the session details, do as follows:</p>
    <ol>
    <li>Go to the <b>Edit session</b> tab.</li>
    <li>Make the necessary changes and click <b>Save</b>.</li>
    </ol>
    </Admonition>
</p>
</li>
<li>On the <b>Meeting link</b> tab, select a meeting platform from the <b>Add meeting link</b> menu.
    <p>
        <Admonition type="note">
        <p>If you haven't selected a meeting platform, <b>BigBlueButton</b> is set as the default platform.</p>
        </Admonition>
    </p>
<table>
  <tr>
    <th>Meeting Platform</th>
    <th>Meeting Information</th>
  </tr>
  <tr>
    <td>BigBlueButton</td>
    <td>No action required.</td>
  </tr>
  <tr>
    <td>Google Meet</td>
    <td>Copy and paste the call link in the <b>Meet link</b> box.</td>
  </tr>
  <tr>
    <td>WhatsApp</td>
    <td>Copy and paste the call link in the <b>WhatsApp</b> box.</td>
  </tr>
  <tr>
    <td>Zoom</td>
    <td>
    Do the following actions: 
    <ol>
        <li>Copy and paste the meeting link in the <b>Zoom link</b> box.</li>
        <li>Copy and paste the meeting ID in the <b>Meeting ID</b> box.</li>
        <li>Copy and paste the passcode in the <b>Passcode</b> box.</li>
    </ol>
    </td>
  </tr>
</table>
</li>
<li>Click <b>Submit</b>. The meeting information is added.
<p>
    <Admonition type="tip">
    <p>You can find the new sessions and track their progress on the Session list page.</p>
    </Admonition>
</p>
</li>
</ol>
