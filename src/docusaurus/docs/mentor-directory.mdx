# Mentor Directory 

The Mentor Directory lists mentor profiles in an alphabetical order. Using the Mentor Directory you can discover mentors in your chosen area of expertise and view their mentor profile.

On the Mentor Profile page, you can do the following actions:

<ul>
<li>View the number of sessions conducted by the mentor.</li>
<li>View the ratings they received for sessions.</li>
<li>Discover the upcoming sessions of the mentor.</li>
<li>Enroll and join sessions.</li>
<li>Share a mentor profile using a link.</li>
</ul>

**To view the Mentor Profile page, do as follows:**

<ol>
<li>To view the Mentor Directory, do any one of the following actions:
<ul>
<li>Do one of the following actions:
<ul>
<li>Select <b>Mentors</b> from the <b>Application</b> menu.</li>
<li>Go to the <b>Application</b> menu <img src={BurgerMenu}/> and select <b>Mentors</b>.</li>
</ul>
<p>
    <img src={PortalAppMenu} />
</p>
</li>
<li>Go to the <b>Mentors</b> tab.
<p>
    <img src={MentorDirectoryIcon} />
</p> 
</li>
</ul>
</li>
<li>Find a mentor and click the mentor profile tile.</li>
<li>Go to the <b>About</b> tab to view the profile details.
<p>
    <img src={PortalMentorProfilePage} />
</p>
</li>
</ol>
