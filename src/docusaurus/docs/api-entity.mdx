# Create Entities

Organizations can define entities that are specific to its domain. The labels assigned to entities appear as options on the session or profile creation pages.

Organization administrators can create new entities to achieve the following:

* Define new entities and map them to an entity type.

* Add new options to the select fields of session or profile creation pages.

:::note Prerequisites

Organization administrators must do the following actions:

  1. After receiving the application's link through email, sign up on the application. See the <a href="aboutorganizationadministrator">Organization Administrator</a> section to learn more.
  2. Log in using `{{url}}/user/v1/account/login` to generate an access_token.

:::

**HTTP Request**

Method: POST

  ```
  {{url}}/mentoring/v1/entity/create

  ```

**Request Body**
