# Getting Started with Mentor

After an organization administrator adds you to the platform and assigns a role (mentor or mentee), you will receive an email notification containing the link to the application. To get started, do one of the following actions:

* Click the link to view the portal on the web browser.

  <p>
    <Admonition type="note">
    <ul>
    <li>If the web portal does not load, update your web browser and retry.</li>
    <li>The supported web browsers are Google Chrome, Firefox, and Safari.</li>
    </ul>
    </Admonition>
  </p>

* On the web browser, select the <b>Install</b> option to install the application on your device.
