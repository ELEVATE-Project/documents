# Deleting a Mentee Account

You can create a request to delete the account. After the request is accepted, you will lose access to your account.

In addition to this, the following actions take place:

* Your enrollment to sessions are cancelled.
* You cannot enroll for a new session.

<Admonition type="tip">
<p>You can reuse the deleted account's credentials if you wish to register again as a new <PartialExample mentored /> user in future.</p>
</Admonition>

**To create a request to delete your account, do as follows:**

<ol>
<li>Do one of the following actions:</li>
<ul>
<li>Select <b>Help</b> from the <b>Application</b> menu.</li>
<li>Go to the <b>Application</b> menu <img src={BurgerMenu}/> and select <b>Help</b>.</li>
</ul>
<li>On the Help page, select <b>Request to delete my account</b> option from the help menu.</li>
<p>
    <img src={PortalMenteeDeleteAccount} />
</p>
<li>Optionally, to specify the reason for deleting the account, enter the reason in the <b>Reason for deleting account</b> box.</li>
<li>Click <b>Delete account</b>.</li>
<li>Click <b>Yes</b> on the confirmation dialog. A request to delete the account is created.</li>
</ol>
