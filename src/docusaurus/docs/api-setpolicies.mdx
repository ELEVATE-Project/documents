# Setting Policies for an Organization

Organization administrators can set policies for their organizations. To learn more about the types of policies that can be configured, see [Policy Architecture](policyarchitecture#policy-architecture).

**HTTP Request**

Method: POST

  ```
  {{url}}/mentoring/v1/org-admin/setOrgPolicies

  ```

**Request Body**

<table>
<tr>
    <th>Payload Parameter</th>
    <th>Data Type</th>
    <th>Description</th>
    <th>Required</th>
</tr>
<tr>
    <td>session_visibility_policy</td>
    <td><p><b>String</b></p><p>Accepted values:<ul><li><i>ALL</i></li><li><i>CURRENT</i></li><li><i>ASSOCIATED</i></li></ul></p></td>
    <td>
    <ul>
    <li>Manage whether your organization’s sessions should be visible to users from other organizations.</li>
    <li>Manage whether your organization’s mentees should be visible to session managers from other organizations.</li>
    </ul>
    </td>
    <td>False</td>    
</tr>
<tr>
    <td>mentor_visibility_policy</td>
    <td><p><b>String</b></p><p>Accepted values:<ul><li><i>ALL</i></li><li><i>CURRENT</i></li><li><i>ASSOCIATED</i></li></ul></p></td>
    <td><p>Manage whether your organization’s mentors should be visible to users from other organizations.</p></td>
    <td>True</td>
</tr>
<tr>
    <td>external_session_visibility_policy</td>
    <td><p><b>String</b></p><p>Accepted values:<ul><li><i>ALL</i></li><li><i>CURRENT</i></li><li><i>ASSOCIATED</i></li></ul></p></td>
    <td>
    <ul>
    <li>Manage whether your organization’s users can view sessions of other organizations.</li>
    <li>Manage whether your organization’s session managers can view mentees of other organizations.</li>
    </ul>
    </td>
    <td>True</td>  
</tr>
<tr>
    <td>external_mentor_visibility_policy</td>
    <td><p><b>String</b></p><p>Accepted values:<ul><li><i>ALL</i></li><li><i>CURRENT</i></li><li><i>ASSOCIATED</i></li></ul></p></td>
    <td><p>Manage whether your organization’s users can view mentors from other organizations.</p></td>
    <td>True</td>  
</tr>
<tr>
    <td>allow_mentor_override</td>
    <td><p><b>Boolean</b></p><p>The default value is <i>false</i>.</p></td>
    <td>Not supported in the current version.</td>
    <td>True</td>  
</tr>
</table>

## Viewing Policies Configured for an Organization

Organization administrators can view the policies configured for their organization.

**HTTP Request**

Method: POST

  ```
  {{url}}/mentoring/v1/org-admin/getOrgPolicies

  ```

**Request Body**

This method does not require a request body.

**Request Header**

|Key|Value|Description|Required|
|---|---|---|---|
|X-auth-token|bearer {{access_token}}|access_token received from the login response.|True|

If successful, this method returns `200 OK` response code.
