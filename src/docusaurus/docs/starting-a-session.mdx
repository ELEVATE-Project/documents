# Starting a Session 

You can host the <PartialExample mentored /> session through your selected meeting platform.

<Admonition type="note">
<ul>
<li>You can start the session only 10 minutes before the scheduled time.</li>
<li>You can start the session anytime within the time allocated for the session.</li>
</ul>
</Admonition>

You can start a session in one of the following ways:

* Using the **Start** button
* Using the session tile
* Using the shared link

## Using the Start Button

<ol>
<li>On your Home page, go the <b>Created sessions</b> tab.</li>
<li>Find the session.</li>
<li>Click <b>Start</b>. The meeting window appears.
<p>
   <Admonition type="tip">    
   <p>If an error occurs, go back to <PartialExample mentored /> and click <b>Start</b>.</p>
   </Admonition>
</p>
<p>
   <img src={PortalStartButton} />
</p>
</li>
</ol>

## Using the Session Details Page

<ol>
<li>On your Home page, go the <b>Created sessions</b> tab.</li>
<li>Click the session tile to view the Session Details page.
<p>
   <img src={PortalNewSession} />
</p>
</li>
<li>Click <b>Start session</b>. The meeting window appears.
<p>
   <img src={PortalStartSessionButton} />
</p>
</li>
</ol>

## Using the Shared Link

<ol>
<li>Click the session link which was shared with your contacts or on social media.</li> 
<li>Click <b>Start session</b>. The meeting window appears.
<p>
   <img src={PortalStartSessionButton} />
</p>
</li>
</ol>

## Usage Guidelines

- Unmute your microphone only when necessary.

- Maintain a neat background and attire before you turn on your webcam. 

- Use respectful language while sharing any information. 

- If you leave a BigBlueButton meeting unintentionally (by pressing the back key on your phone or closing the meeting tab), go back to <PartialExample mentored /> and click <b>Start</b>.
