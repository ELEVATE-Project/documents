# Uploading Users to the Application

The organization administrator can upload the list of users to the application using a CSV file to:

* Invite new users with preassigned roles (mentor and mentee).

* Invite registered users to an organization. After uploading the list of registered users, they are assigned an organization.

* Allow an existing user to host sessions by changing the role to mentor.

**To upload users to the application, do as follows:**

1. Go to the <b>Application</b> menu ![burger menu](media/burgermenu-icon.png) and select **Admin Workspace**.

2. On the Admin Workspace page, click **Manage user**.

3. On the **Bulk Upload** tab, click **Download sample CSV** to download the sample CSV file.

4. Open the downloaded CSV file and clear the sample details.

    ![](media/orgadmin-bulkupload.png)

5. Enter the following user information:

    * Email ID
    * Assigned role
 
6. On the **Bulk Upload** tab, click **Upload CSV** and upload the file containing the list of users.

    <Admonition type="tip">
    <p>You will receive an email notification containing a CSV file with the following information about the listed users:</p>
    <ul>
    <li>Assigned user ID</li>
    <li>Whether the user uploaded to the application already exists</li>
    </ul>
    </Admonition>
