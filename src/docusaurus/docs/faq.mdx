# FAQ

<details>
<summary>What does <PartialExample mentored /> application offer?</summary> 
<p><PartialExample mentored /> is an open source solution that creates a mentoring network and make experts discoverable to mentees to connect and seek guidance for professional development.</p>

<p>To learn more, see <a href="intro">Welcome to <PartialExample mentored /></a>.</p>

</details>

<details>
<summary>How can mentors and mentees use <PartialExample mentored />?</summary>

* Mentors can create sessions and join other sessions. 
* Mentees can join sessions. 

To learn more, see <a href="using-the-application">About the <PartialExample mentored /> Application</a>.
</details>

<details>
<summary>How can organizations manage users (mentors and mentees) on the platform?</summary>

Users with Organization Administrator privileges can manage users in the following ways:

* Upload the list of users with preassigned roles to the platform. 
* Manage users' requests to become a mentor. 

To learn more, see the following topics:

* <a href="bulkupload">Uploading Users to the Application</a>
* <a href="managementorrequests">Managing Requests to Become a Mentor</a>

</details>

<details>
<summary>How can organizations manage sessions on the platform?</summary>

Session managers can create Public or Private sessions and do the following actions:

* Assign the sessions to respective mentors. The assigned sessions are hosted by mentors during the scheduled time.
* Invite mentees to the sessions.
* Conduct session management activities such as updating the session and meeting details and tracking the session's status and mentee enrollments.

To learn more, see the following topics:

* <a href="createsession-sessionmanager">Creating Sessions for an Organization</a>
* <a href="manageassignedsession">Managing Sessions Assigned to Mentors</a>

</details>

<summary>How can I sign up on <PartialExample mentored />?</summary>

After receiving an email invite containing the application's link, new users can sign up to create their accounts. If you have received a session or mentor profile link from your contacts, you can use the link to sign up as a mentee.
<p>
   <Admonition type="tip"> 
   <p>By default, everyone on <PartialExample mentored /> has mentee privileges to connect with mentors and seek guidance (through mentoring sessions).</p>

   * Notice a login from an unfamiliar device on the Login activity page.

<details>
<summary>What is the duration of each session?</summary>

The minimum duration of a session should be 30 minutes. 

To learn more, see <a href="setting-up-a-session">Setting up a Session</a>.
</details>

<details>
<summary>How many sessions can I create?</summary>

The platform does not apply any limits on the number of sessions that a mentor can host.

</details>

Mentors can host their sessions using any one of the following meeting platforms:

* WhatsApp
* Zoom

<details>
<summary>Is there a limit on the number of participants who can enroll for a session?</summary>

You cannot enroll for a session after the enrollment limit is reached. If you try to enroll after this limit is reached, a *Session seats full* message appears.

To learn more, see <a href="enrolling-for-a-session">Enrolling for a Session</a>.

</details>

<details>
<summary>Can I cancel my enrollment?</summary>

You can cancel your enrollment if the mentor has not started the session.

</details>

To learn more, see <a href="joining-a-session">Joining a Session</a>.
