# Managing Requests to be a Mentor

After mentees raise a request to be a mentor, the organization administrator can review their profile details and accept or reject the request.

**To accept requests to be a mentor, do as follows:**

1. Go to the <b>Application</b> menu ![burger menu](media/burgermenu-icon.png) and select **Admin Workspace**.

2. On the Admin Workspace page, click **Manage user**. The Manage user page appears.

    ![](media/orgadmin-managerequests.png)
   
4. On the Manage Request tab, do one of the following actions:

    * To review the profile details, click **View** ![](media/orgadmin-viewmentordetails.png).

    * To accept the request, do as follows:
        1. Click **Accept** ![](media/orgadmin-acceptmentor.png).
        2. Click **Yes** in the confirmation dialog.

    * To reject the request, do as follows:
        1. Click **Reject** ![](media/orgadmin-rejectmentor.png).
        2. Click **Yes** in the confirmation dialog.
