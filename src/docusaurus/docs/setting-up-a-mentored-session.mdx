# Setting up a Session

You can create and manage mentoring sessions.

**To create a session, do as follows:**

<Tabs>
<TabItem value="mentored mobile application" label="Mobile App" default>
<ol>
<li>On the Home page, go to the <b>Created sessions</b> tab.</li>
<li>Tap <b>Create session</b>.
<p>
    <img src={SessionDetails} />
</p>
</li>
<li>To update your profile picture, tap <b>Add image</b> and do one of the following actions:
    <ul>
    <li>To take a picture, select <b>Take Photo</b>.</li>
    <li>To upload an image, select <b>Choose from library</b>.</li>
    <li>To remove the existing image, select <b>Remove current photo</b>.</li>
    </ul>
</li>
<li>Enter the session information in the <b>Session title</b> and <b>Description</b> box.</li>
<li>To schedule a session, do as follows:
    <ol>
    <li>Tap the <b>Start date</b> or <b>End date</b> box.</li>
    <li>Select the date and time from the <b>Set date and time</b> menu and tap <b>Set</b>.</li>
    </ol>
<p>  
    <Admonition type="note">
    <p>The session duration should be at least 30 minutes.</p>
    </Admonition>
</p>
</li>
<li>Select an option from the following boxes:
    <ul>
    <li><b>Recommended for</b></li>
    <li><b>Categories</b></li>
    <li><b>Medium</b></li>
    </ul>
<p>
    <Admonition type="tip">
    <ul>
    <li>To cancel the selection, tap the option again.</li>
    <li>To add a new option, do as follows:
        <ol>
        <li>Tap <b>+ Add</b> and enter the new option.</li>
        <li>Tap <b>Ok</b>.</li>
        </ol>
    </li>
    <li>To select all the options, turn on the toggle.</li>
    </ul>
    </Admonition>
</p>
<p>
    <img src={SessionDetailsOptions} />  
</p> 
</li>
<li>If you want to add a profile picture, tap <b>Add image</b>. Select <b>Take Photo</b> or <b>Choose from library</b>.</li>
<li>Tap <b>Publish and add link</b>. The session is published and the <b>Meeting Link</b> tab is enabled.
<p>
    <img src={AddMeetingLink} />  
</p>
</li>
<li>To select a meeting platform other than <b>BigBlueButton</b> on the <b>Meeting Link</b> tab, select a meeting platform from the <b>Add meeting link</b> menu.
<p>
    <Admonition type="tip">
    <p><b>BigBlueButton</b> is the default meeting platform.</p>
    </Admonition>
</p>
</li>
<li>To add the details about the meeting platform, do as follows:
    <ul>
    <li>If you have selected Google Meet as the meeting platform, paste the meeting link.</li>
    <li>If you have selected WhatsApp as the meeting platform, paste the call link.</li>
    <li>If you have selected Zoom as the meeting platform, paste the <b>Meeting ID</b> and <b>Password</b>.</li>
    </ul>
</li>
<li>Tap <b>Submit</b>. The meeting information is added.
<p>
    <Admonition type="tip">
    <ul>
    <li>You can find the new sessions on the <b>Created session</b> tab on the Home page.</li>
    <li>To create more sessions, click the <b>Click here to create a new session</b> button and repeat steps 1 to 10.</li>
    </ul>
    </Admonition>
</p>
</li>
</ol>
</TabItem>
<TabItem value="mentored portal" label="Web Portal">
<ol>
<li>Go to the <b>Created sessions</b> tab.</li>
<li>Click the <b>Click here to create a new session</b> button.
<p>
    <img src={PortalSessionDetails} />
</p>
</li>
<li>To add a session image, click <b>Add image</b> and select the image.</li>
<li>Enter the session information in the <b>Session title</b> and <b>Description</b> box.</li>
<li>To schedule a session, do as follows:
    <ol>
    <li>Click the <b>Start date</b> or <b>End date</b> box.</li>
    <li>Select the date and time from the menu and click <b>Set</b>.</li>
    </ol>
<p> 
    <Admonition type="note">
    <ul>
    <li>The session duration should be at least 30 minutes.</li>
    <li>While setting the end time, set an extra time of at least 10 minutes to account for any delays during the session.</li>
    </ul>
    </Admonition>
</p>
</li>
<li>Select an option in the following boxes:
<ul>
<li><b>Recommended for</b></li>
<li><b>Categories</b></li>
<li><b>Medium</b></li>
</ul>
<p>
    <Admonition type="tip">
    <ul>
    <li>To cancel the selection, click the option again.</li>
    <li>To add a new option, do as follows:
        <ol>
        <li>Click <b>+ Add new</b> and enter the new option.</li>
        <li>Click <b>Ok</b>.</li>
        </ol>
    </li>
    <li>To select all the options, turn on the toggle.</li>
    </ul>
    </Admonition>
</p>
<p>
    <img src={PortalSessionDetailsOptions} />
</p>
</li>
<li>Click <b>Publish and add link</b>. The session is published and the <b>Meeting Link</b> tab is enabled.
<p>
    <img src={PortalAddMeetingLink} />  
</p>
</li>
<li>To select a meeting platform other than <b>BigBlueButton</b> on the <b>Meeting Link</b> tab, select a meeting platform from the <b>Add meeting link</b> menu.
<p>
    <Admonition type="tip">
    <p><b>BigBlueButton</b> is the default meeting platform.</p>
    </Admonition>
</p>
</li>
<li>To add the details about the meeting platform, do as follows:
    <ul>
    <li>If you have selected Google Meet as the meeting platform, paste the meeting link.</li>
    <li>If you have selected WhatsApp as the meeting platform, paste the call link.
    <p>
        <Admonition type="note">
        <p>If you select WhatsApp as the meeting platform, you can host the sessions only through the WhatsApp mobile app.</p>
        </Admonition>
    </p>
    </li>
    <li>If you have selected Zoom as the meeting platform, paste the <b>Meeting ID</b> and <b>Password</b>.</li>
    </ul>
</li>
<li>Click <b>Submit</b>. The meeting information is added.
<p>
    <Admonition type="tip">
    <ul>
    <li>You can find the new sessions on the <b>Created session</b> tab.</li>
    <li>To create more sessions, click the <b>Click here to create a new session</b> button and repeat steps 1 to 10.</li>
    </ul>
    </Admonition>
</p>
<p>
    <img src={PortalNewSession} />
</p>
</li>
</ol>
</TabItem>
</Tabs>

## Editing a Session

<Admonition type="note">
<p>If you want to edit the session details, you must do so at least 10 minutes before the start time.</p>
</Admonition>

<Tabs>
<TabItem value="mentored mobile application" label="Mobile App" default>
<ol>
<li>To edit the session details, tap the session tile to view the Session Details page.</li>
<li>Tap <b>Edit</b>.
<p>
    <img src={EditAndDeleteSession} />
</p>
</li>
<li>To edit the session details and the profile picture on the Edit session details page, go to the <b>Edit session</b> tab and do as follows:
    <ol>
    <li>Change the session details.</li>
    <li>To update your profile picture, tap <b>Add image</b> and do one of the following actions:
        <ul>
        <li>To take a picture, select <b>Take Photo</b>.</li>
        <li>To upload an image, select <b>Choose from library</b>.</li>
        <li>To remove the existing image, select <b>Remove current photo</b>.</li>
        </ul>
    </li>
    </ol>
</li>
<li>Tap <b>Save</b>. The session details are updated.</li>
<li>To edit the meeting platform and the meeting information, go to the <b>Meeting link</b> tab and make the necessary changes.</li>
<li>Tap <b>Submit</b>. The meeting information is updated.</li>
</ol>
</TabItem>
<TabItem value="mentored portal" label="Web Portal">
<ol>
<li>Do one of the following actions: 
    <ul>
    <li>Click the session tile to view the Session Details page.</li>
    <li>Click the <b>View details</b> link to view the Session Details page.</li>
    </ul>
</li>
<li>Click <b>Edit session</b>.
<p>
    <img src={PortalEditAndDeleteSession} />
</p>
</li>
<li>To edit the session details and the profile picture on the Edit session details page, go to the <b>Edit session</b> tab and do as follows:
    <ol>
    <li>Change the session details.</li>
    <li>To update your profile picture, click <b>Add image</b> and do one of the following actions:
        <ul>
        <li>To take a picture, select <b>Take Photo</b>.</li>
        <li>To upload an image, select <b>Choose from library</b>.</li>
        <li>To remove the existing image, select <b>Remove current photo</b>.</li>
        </ul>
    </li>
    </ol>
</li>
<li>Click <b>Save</b>. The session details are updated.</li>
<li>To edit the meeting platform and the meeting information, go to the <b>Meeting link</b> tab and make the necessary changes.</li>
<li>Click <b>Submit</b>. The meeting information is updated.</li>
</ol>
</TabItem>
</Tabs>

## Deleting a Session

<Admonition type="note">
<p>If you want to delete the session, you must do so at least 10 minutes before the start time.</p>
</Admonition>

<Tabs>
<TabItem value="mentored mobile application" label="Mobile App" default>
<p>To delete the session, tap the session tile and tap <b>Delete</b>. The session is cancelled and is removed from the <b>Created sessions</b> tab of the Home page.</p>
<p>
    <img src={EditAndDeleteSession} />
</p>
</TabItem>
<TabItem value="mentored portal" label="Web Portal">
    <p><b>To delete a session, do as follows:</b></p>
    <ol>
        <li>Do one of the following actions:
            <ul>
                <li>Click the session tile to view the Session details page.</li>
                <li>Click the <b>View details</b> link to view the Session details page.</li>
            </ul>
        </li>
        <li>Click <b>Delete session</b>.
        <p>
            <img src={PortalEditAndDeleteSession} />
        </p>
        </li>
        <li>Click <b>Yes, delete</b> on the confirmation dialog. The session is cancelled and is removed
        from the Upcoming sessions section of the <b>Created sessions</b> tab.</li>
    </ol>
</TabItem>
</Tabs>
