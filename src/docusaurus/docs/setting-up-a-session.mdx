# Setting up a Session

To share your skills and knowledge with your mentees, you can create a <PartialExample mentored /> session and add requisite details such as:

* The topic for the session
* The session schedule
* A creative graphic or image to attract the attention of mentees
* The meeting platform

**To create a session, do as follows:**

<ol>
<li>On the Home page, go to <b>My mentoring sessions</b> tab.</li>
<li>Click <b>Create session</b>. The Create a new session page appears.
<p>
    <img src={PortalSessionDetails} />
</p>
</li>
<li>Optionally, to add a session image, click <b>Add image</b> and do one of the following actions:
    <ul>
    <li>To take a picture, select <b>Take Photo</b>.</li>
    <li>To upload an image, select <b>Choose from library</b>.
        <p>  
        <Admonition type="tip">
        <p>Select an image that is relevant to the session topic.</p>
        </Admonition>
        </p>
    </li>
    </ul>
</li>
<li>Enter the session information in the <b>Session title</b> and <b>Description</b> boxes.</li>
<li>Mentors with Session Manager privileges should do the following actions:
    <ol>
    <li>To select the type of session, do as follows:
        <ol>
        <li>Click the <b>Session type</b> box.</li>
        <li>Select the session type from the <b>Session type</b> dropdown and click <b>Ok</b>.</li>
        </ol>
        <p>  
            <Admonition type="note">
            <p>You cannot change the session type after publishing the session.</p>
            </Admonition>
        </p>
    </li>
    <li>To add your name in the <b>Add mentor</b> box, do as follows:
        <ol>
        <li>Click <b>+ Add mentor</b>. The Mentor list window appears.</li>
        <li>Find your name from the list and click <b>Add</b>.
            <p>
            <img src={MentorListWindow} />
            </p>
        </li>
        </ol>
        <p>  
            <Admonition type="tip">
            <p>From the Mentor or Mentee list window, you can find users by:</p>
            <ul>
            <li>Searching for the user in the <b>Search</b> box.</li>
            <li>Using filters such as the user's organization or designation.</li>
            <li>Changing the number of users listed on the page using the <b>Items per page</b> dropdown.</li>
            </ul>
            </Admonition>
        </p>
    </li>
    <li>To invite mentees, do as follows:
        <ol>
        <li>Click <b>+ Add mentee</b>. The Mentee list window appears.</li>
        <li>Find a mentee from the list and click <b>Add</b>.
            <p>
            <img src={MenteeListWindow} />
            </p>
        </li>
        </ol>
        <p>  
            <Admonition type="tip">
            <p>It is optional to invite mentees to a Public session.</p>
            </Admonition>
        </p>
    </li>
    </ol>
</li>
<li>To schedule a session, do as follows:
    <ol>
    <li>Click the <b>Start date</b> and <b>End date</b> boxes.</li>
    <li>Select the date and time from the date and time picker.</li>
    <li>Click <img src={SelectDateAndTime} />.</li>
    </ol>
<p>  
    <Admonition type="note">
    <p>The session duration should be at least 30 minutes.</p>
    </Admonition>
</p>
</li>
<li>Select an option from the following boxes:
    <ul>
    <li><b>Recommended for</b></li>
    <li><b>Categories</b></li>
    <li><b>Medium</b></li>
    </ul>
<p>
    <Admonition type="tip">
    <ul>
    <li>To cancel the selection, click the option again.</li>
    <li>To add a new option, do as follows:
        <ol>
        <li>Click <b>+ Add</b> and enter the new option.</li>
        <li>Click <b>Ok</b>.</li>
        </ol>
    </li>
    <li>To select all the options, turn on the toggle.</li>
    </ul>
    </Admonition>
</p>
<p>
    <img src={PortalSessionDetailsOptions} />  
</p> 
</li>
<li>Click <b>Publish and add link</b>. The session is published on the platform and participants can now view your session.
<p>
    <Admonition type="note">
    <p>You must set the meeting platform on the <b>Meeting Link</b> tab.</p>
    </Admonition>
</p>
<p>
    <img src={PortalAddMeetingLink} />  
</p>
<p>
    <Admonition type="tip">
    <p>To go back and edit the session details, do as follows:</p>
    <ol>
    <li>Go to the <b>Edit session</b> tab.</li>
    <li>Make the necessary changes and click <b>Save</b>.</li>
    </ol>
    </Admonition>
</p>
</li>
<li>On the <b>Meeting link</b> tab, select a meeting platform from the <b>Add meeting link</b> menu.
    <p>
        <Admonition type="note">
        <p>If you haven't selected a meeting platform, <b>BigBlueButton</b> is set as the default platform.</p>
        </Admonition>
    </p>
<table>
  <tr>
    <th>Meeting Platform</th>
    <th>Meeting Information</th>
  </tr>
  <tr>
    <td>BigBlueButton</td>
    <td>No action required.</td>
  </tr>
  <tr>
    <td>Google Meet</td>
    <td>Copy and paste the call link in the <b>Meet link</b> box.</td>
  </tr>
  <tr>
    <td>WhatsApp</td>
    <td>Copy and paste the call link in the <b>WhatsApp</b> box.</td>
  </tr>
  <tr>
    <td>Zoom</td>
    <td>
    Do the following actions: 
    <ol>
        <li>Copy and paste the meeting link in the <b>Zoom link</b> box.</li>
        <li>Copy and paste the meeting ID in the <b>Meeting ID</b> box.</li>
        <li>Copy and paste the passcode in the <b>Passcode</b> box.</li>
    </ol>
    </td>
  </tr>
</table>
</li>
<li>Click <b>Submit</b>. The meeting information is added.
<p>
    <Admonition type="tip">
    <p>You can find the new session on <b>My mentoring sessions</b> tab on the Home page.</p>
    <p>On <b>My mentoring sessions</b> tab, you can view your upcoming and completed sessions.</p>
    </Admonition>
</p>
</li>
</ol>

## Tracking the Enrollments

1. On <b>My mentoring sessions</b> tab, click the session tile.

    <p>
        <img src={PortalNewSession} />
    </p>

2.  In the **Mentee count** section of the Session Details page, do one of the following actions:

    * View the number of enrollments.

        <p>
        <img src={PortalEditAndDeleteSession} />
        </p>

    * To view the list of mentees who were enrolled or invited for the session, click **View list**. 

        <p>
        <Admonition type="tip">
        <p>Click the <b>Download</b> link to download a CSV of list.</p>
        </Admonition>
        </p>

        ![](media/menteeenrollmentlist.png)

## Editing a Session

1. On <b>My mentoring sessions</b> tab, click the session tile.

    <p>
        <img src={PortalNewSession} />
    </p>

2. On the Session Details page, click <b>Edit</b>. The Edit session details page appears.

    <p>
        <img src={PortalEditAndDeleteSession} />
    </p>

### Editing the Session Details

<p>
<Admonition type="note">
<p>You can edit the session details only until the start of the session.</p>
</Admonition>
</p>
<ol>
<li>On the Edit session details page, go to the <b>Edit session</b> tab.</li>
<li>Make the necessary changes to the session information.</li>
<li>Optionally, to update your session image, click <b>Add image</b> and do one of the following actions:
    <ul>
    <li>To take a picture, select <b>Take Photo</b>.</li>
    <li>To upload an image, select <b>Choose from library</b>.</li>
    <li>To remove the existing image, select <b>Remove current photo</b>.</li>
    </ul>
<p>  
    <Admonition type="tip">
    <p>Select an image that is relevant to the session topic.</p>
    </Admonition>
</p>
</li>
<li>Click <b>Save</b>. The session details are updated.</li>
</ol>

### Editing the Meeting Information
<p>
<Admonition type="tip">
<p>After starting the session, you can go back to <PartialExample mentored /> and edit the meeting information. The session that you have started appears as <i>Live</i> on the <b>My mentoring sessions</b> tab.</p>
</Admonition>
</p>
<ol>
<li>On the Edit Session Details page, go to the <b>Meeting link</b> tab and make the necessary changes.</li>
<li>Click <b>Submit</b>. The meeting information is updated.</li>
</ol>

## Deleting a Session

<Admonition type="note">
<p>You can delete the session only until the start of the session.</p>
</Admonition>

**To delete the session, do as follows:**

1. On <b>My mentoring sessions</b> tab, click the session tile.

    <p>
        <img src={PortalNewSession} />
    </p>

2. Click the session tile and click <b>Delete</b>.

    <p>
        <img src={PortalEditAndDeleteSession} />
    </p>

3. Click <b>Yes delete</b> on the confirmation dialog. The session is cancelled and is removed from the <b>My mentoring sessions</b> tab of the Home page.
