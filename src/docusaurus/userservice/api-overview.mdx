# Overview

You can set up organizations only using the REST APIs of User service service.

Using the REST APIs of User service, you can do the following:

  * Create a System Administrator account.

  * Log in to the System Administrator account.

  * Log in to the Organization Administrator account.

  * Create and manage organizations.

  * Assign an organization administrator.

  * Manage users at an organization.

## Accessing the API Collection

To access the APIs on Postman, do any one of the following actions:

* Import the JSON and environment file from <a href="https://github.com/ELEVATE-Project/user/tree/master/src/api-doc">https://github.com/ELEVATE-Project/user/tree/master/src/api-doc</a>

* Import the raw URL from <a href="https://raw.githubusercontent.com/ELEVATE-Project/user/master/src/api-doc/MentorED-Users.postman_collection.json">https://raw.githubusercontent.com/ELEVATE-Project/user/master/src/api-doc/MentorED-Users.postman_collection.json</a>
