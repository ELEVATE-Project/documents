# Manage Organizations

After creating organizations, administrators can manage the organization's details and users.

## How System Administrators Can Manage Organizations

<table>
  <tr>
    <th>API Reference</th>
    <th>Description</th>
  </tr>
    <tr>
    <td><a href="https://dev.elevate-apis.shikshalokam.org/user/api-doc#tag/Organization/paths/~1user~1v1~1organization~1list/get">Get Organization Details</a></td>
    <td>Before updating or deactivating organizations, system administrators must obtain organization's details such as organization ID.</td>
  </tr>
  <tr>
    <td><a href="https://dev.elevate-apis.shikshalokam.org/user/api-doc#tag/Organization/operation/updateOrganization">Update Organizations</a></td>
    <td><p>System administrators can update an organization's details such as name and description and specify its associated organizations.</p></td>
  </tr>
  <tr>
    <td><a href="https://dev.elevate-apis.shikshalokam.org/user/api-doc#tag/Admin-APIs/paths/~1user~1v1~1admin~1deactivateOrg~1%7Bid%7D/post">Deactivate Organizations</a></td>
    <td>System administrators can deactivate organizations that no longer exists.</td>
  </tr>
</table>
