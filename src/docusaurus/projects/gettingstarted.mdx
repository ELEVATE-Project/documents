# Getting Started with Projects

After receiving the application's link, you can get started using one of the following ways:

* Click the link to view the portal on the web browser.

  <p>
    <Admonition type="note">
    <ul>
    <li>If the web portal does not load, update your web browser and retry.</li>
    <li>The supported web browsers are Google Chrome, Firefox, and Safari.</li>
    </ul>
    </Admonition>
  </p>

* To install the application on your device, do one of the following actions:

    * Select the <b>Install</b> option on the web browser.
    * Select the <b>Add to Home screen</b> option on the web browser.
