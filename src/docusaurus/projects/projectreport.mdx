# Accessing Project Reports

On the reports page, you can view the following insights about the project:

* Project overview: View the quantitative statistics of the projects that have been started, ongoing, or submitted. 
* Task overview: View the task completion status.
* Category overview: View insights about your completed projects, grouped by category.

**To access reports, do as follows:**

1. Click the **Reports** tile.

2. Go to **Improvement Project Reports**. The Project Reports page appears.

3. Review you project summary.

    <p>
    <Admonition type="tip">
    <ul>
    <li>To view the project's progress for a specific time frame, select <b>Monthly</b>, <b>Weekly</b>, or <b>Quarterly</b> using the Timeframe dropdown.</li>
    <li>To download the report as a PDF file, go to <b>More options</b> <img src={MoreOptions} /> on the top-right corner and select <b>Download</b>.</li>
    </ul> 
    </Admonition>
    </p>

## Sharing Reports

To can share the project report as a PDF, do as follows:

1. Go to <b>More options</b> ![](media/moreoptions.png) on the top-right corner and select <b>Share</b>.

2. To obtain the shareable link, click **Copy**.

3. Share the link with your peers or superiors.

    <Admonition type="tip">
    <p>After clicking on the shared link, they can download a PDF file of the project report.</p>
    </Admonition>
