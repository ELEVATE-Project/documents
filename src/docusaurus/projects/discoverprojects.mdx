# Discovering Projects

You can explore and take up improvement projects that match your interests and expertise.

You can discover projects using any one of the following ways:

* From the **Library**, select a project tile. The Project details page appears. To learn more about viewing the Library, see <a href="projectlibrary">Project Library</a>.

* Click the shared project link.

* Scan the shared project QR code using the QR code scanner available on the mobile app.

    <Admonition type="info">
    <p>For more information on installing the mobile app, see <a href="gettingstarted">Getting Started with Projects</a>.</p>
    </Admonition>

## Viewing the Project's Details

From the list of projects, select a project of your choice. You can view the following details on the **Project details** tab:

* Project's information such as objective, recommended for, and duration.

* Learning resources such as links or files that provide more clarity and understanding about the project.

## Starting Your Discovered Projects

1. After reading the project's details, click **Start Improvement** to initiate the project. 

2. To choose whether to share the project's information with the administrator when submitting it, do one of the following actions:

    * Click **Do not share** if you don't wish to share the project's details.
    * Do as follows:

        1. Read the organization's Terms of Use and Content policy and select the checkbox to confirm that the project's information complies with the policy.

        2. Click **Share**.

<Admonition type="info">
<p>After starting, The project is listed in the <b>Discovered by me</b> tab of the Project page. To learn more about viewing you discovered projects, see <a href="viewprojects">Viewing Projects</a>.</p>
</Admonition>

### Updating Project Sharing Preferences

After starting the project, you can update your preference on whether to share the project's information with the administrator when you submit it.

**To update the preference, do as follows:**

1. On the **Project details** tab, view the Share project details section.

2. Select a preference and click **Save**.
