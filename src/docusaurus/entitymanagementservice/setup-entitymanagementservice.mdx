# Setting up Entity Management Service

<table>
<tr>
<th>Function</th>
<th>Description</th>
</tr>
<tr>
<td>Create and manage entity types</td>
<td>System administrators can define different entity types under which entities are categorized. <b>State</b>, <b>District</b>, and <b>Block</b> are examples of entity types.</td>
</tr>
<tr>
<td>Create and manage entities</td>
<td>Under each entity type, system administrators can create multiple entities. For instance, <b>State</b> (entity type) can include <b>Karnataka</b> as an entity.</td>
</tr>
<tr>
<td>Create hierarchies by defining parent and child entities</td>
<td>System administrators can establish hierarchies between entities of different entity types by mapping child entities to parent entities. For example, the child entity <b>Bangalore</b> (of the entity type <b>District</b>) can be mapped to the parent entity <b>Karnataka</b> (of the entity type <b>State</b>).</td>
</tr>
</table>

See [Setting up Entity Management Service](https://github.com/ELEVATE-Project/entity-management/blob/main/README.md).
