# Reporting Sign up and Log in Issues

You can write to the support team about any problems you faced while signing up or logging in to <PartialExample mentored />.

**To report issues related to signing up and logging in, do as follows:**

<Tabs>
<TabItem value="mentored mobile application" label="Mobile App" default>
<ol>
<li>On the Login page, tap the <b>Having trouble in logging in/signing up? Write to us.</b> link.
<p>
    <img src={LoginPage} />
</p>
</li>
<li>On the pop-up window displaying the email applications, select an email application.</li>
<li>Send an email describing the problems you had faced while signing up or logging in.</li>
</ol>   
</TabItem>
<TabItem value="mentored portal" label="Web Portal">
<ol>
<li>On the Login page, click the <b>Having trouble in logging in/signing up? Write to us.</b> link.
<p>
    <img src={PortalLoginPage} />
</p>
</li>
<li>On the pop-up window displaying the email applications, select an email application.</li>
<li>Send an email describing the problems you had faced while signing up or logging in.</li>
</ol>  
</TabItem>
</Tabs>
