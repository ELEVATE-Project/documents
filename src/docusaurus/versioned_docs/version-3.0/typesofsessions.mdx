# Types of Sessions on Mentor

Users with Session Manager privileges can create, host or assign Public and Private sessions.

<table>
<tr>
<th>Types of Sessions</th>
<th>Description</th>
</tr>
<tr>
<td>Public</td>
<td>
<ul>
<li>Mentees can discover and enroll for the sessions.</li>
<li>Optionally, apart from the enrolled mentees, session managers can also invite mentees to the session.</li>
<li>Mentors and mentees can share the session links to with their contacts or on social media.</li>
</ul>
</td>
</tr>
<tr>
<td>Private</td>
<td>
<ul>
<li>Only invited mentees can join a Private session.</li>
<li>To facilitate one-on-one mentoring, session managers can set up Private sessions consisting of a mentor and an invited mentee.</li>
</ul>
</td>
</tr>
</table>
