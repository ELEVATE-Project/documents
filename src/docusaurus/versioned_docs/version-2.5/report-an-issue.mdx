# Reporting an Issue

You can write to the support team about any problems you faced while using the application.

**To report an issue, do as follows:**

<ol>
<li>Do one of the following actions:</li>
<ul>
<li>Select <b>Help</b> from the <b>Application</b> menu.</li>
<li>Go to the <b>Application</b> menu <img src={BurgerMenu}/> and select <b>Help</b>.</li>
</ul>
<li>On the Help page, select <b>Report an issue</b> option from the help menu.</li>
<li>Enter the details of the issue you faced in the <b>Report an issue</b> box.</li>
<p>
    <img src={PortalHelp} />
</p>
<li>Click <b>Submit</b>. Your issue is reported successfully.</li>
</ol>
