# Creating and Managing a Mentee Profile

On the Profile page, you can do the following actions:

<ul>
<li>Update your profile.</li>
<li>View the number of sessions attended.</li>
</ul>

## Viewing the Profile Page

To view the Profile page, do one of the following actions:

<ul>
<li>Do one of the following actions:
<ul>
<li>Select <b>Profile</b> from the <b>Application</b> menu.</li>
<li>Go to the <b>Application</b> menu <img src={BurgerMenu}/> and select <b>Profile</b>.</li>
</ul>
<p>
    <img src={PortalAppMenu} />
</p>
</li>
<li>Go to the <b>Profile</b> tab.
<p>
    <img src={ProfileIcon} />
</p>
</li>
</ul>

## Updating your Profile

<Admonition type="note"> 
<p>You can use the application only after updating your profile.</p>
</Admonition> 

**To update your profile, do as follows:**

<ol>
<li>On the Profile page, click <b>Edit profile</b>.
<p>
    <img src={PortalEditMenteeProfile} />
</p>
</li>
<li>Optionally, to update your profile picture, do any one of the following actions: 
<ul>
<li>Click <b>Add image</b> and select the image.</li>
<li>To remove an existing image, click <b>Remove image</b>.</li>
</ul>
</li>
<li>Enter the information in the following boxes:
    <ul>
    <li><b>Your experience in years</b></li>
    <li><b>Tell us about yourself</b></li>
    <li><b>Education qualification</b></li>
    </ul>
</li>
<li>To select your location, click the <b>Select your location</b> box and select the location from the menu.</li>
<li>Select an option in the following boxes:
    <ul>
    <li><b>Your role</b></li>
    <li><b>Your expertise</b></li>
    <li><b>Languages</b></li>
    </ul>
<p>
    <Admonition type="tip">
    <ul>
    <li>To cancel the selection, click the option again.</li>
    <li>To add a new option, do as follows:
        <ol>
        <li>Click <b>+ Add new</b> and enter the new option.</li>
        <li>Click <b>Ok</b>.</li>
        </ol>
    </li>
    <li>To select all the options, turn on the toggle.</li>
    </ul>
    </Admonition>
</p>
<p>
    <img src={PortalCreateProfileOptions} />
</p>
</li>
<li>Click <b>Submit</b>. The profile details are updated.</li>
</ol>
