# Joining a Session

After enrolling, you can join when the mentor starts the session.

<Admonition type="note">
You can join the session anytime within the time allocated for the session.
</Admonition>

<p>You can join a session in one of the following ways:</p>
<ul>
<li>Using the <b>Enrolled sessions</b> tab</li>
<li>Using the <b>All sessions</b> tab</li>
<li>Using the mentor profile</li>
<li>Using the Session Details page</li>
<li>Using the <b>Search</b> bar</li>
</ul>

## Using the Enrolled Sessions Tab

<ol>
<li>On the Home page, go to the <b>Enrolled sessions</b> tab.</li>
<li>Find the session.</li>
<li>Click <b>Join</b>.</li>
<p>
    <Admonition type="tip">    
    <p>If an error occurs, go back to <PartialExample mentored /> and click <b>Join</b>.</p>
    </Admonition>
</p>
<p>
    <img src={PortalJoinButton} />
</p>
<JoinStepPortal />
</ol>

## Using the All Sessions Tab

<ol>   
<li>On the Home page, go to the <b>All Sessions</b> tab.</li> 
<li>Find an enrolled session.</li>
<li>Click <b>Join</b>.</li>
<p>
    <img src={PortalJoinButton} />
</p>
<JoinStepPortal />
</ol>

## Using the Session Details Page

<ol>
<li>Find an enrolled session from the <b>All Sessions</b> tab, <b>Enrolled Sessions</b> tab, or the Mentor Profile page.</li>
<li>Click the session tile to view the Session Details page.</li>
<p>
    <img src={PortalSessionTileJoin} />
</p>
<li>Click <b>Join</b>.</li>
<p>
    <img src={PortalSessionDetailsJoin} />
</p>
<JoinStepPortal />
</ol>

## Using the Search bar

<ol>
<li>Click the <b>Sessions</b> tab.</li>
<p>
    <img src={PortalSearchSessions} />
</p>
<li>Type the session name and click the <b>Search</b> button.</li>
<li>Click <b>Join</b>.</li>
<JoinStepPortal />
</ol>

## Usage Guidelines

- Unmute your microphone only when necessary.

- Maintain a neat background and attire before you turn on your webcam. 

- Use respectful language while sharing any information.

- If you leave a BigBlueButton meeting unintentionally (by pressing the back key on your phone or closing the meeting tab), go back to <PartialExample mentored /> and click <b>Join</b>.
