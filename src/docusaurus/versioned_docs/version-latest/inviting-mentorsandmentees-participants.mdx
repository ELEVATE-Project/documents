# Sharing Session Invites to Your Contacts

You can invite participants to a session by sharing a session link.

<Tabs>
<TabItem value="mentored mobile application" label="Mobile App" default>
<p>You can share a session link in one of the following ways:</p>
<ul>
<li>Using the <b>All sessions</b> tab</li>
<li>Using the <b>Enrolled sessions</b> tab</li>
</ul>
<h2>Using the All Sessions Tab</h2>
<ol>
<li>On the Home page, go to the <b>All sessions</b> tab.</li>
<li>Find an upcoming session. </li>
<li>Tap the session tile and tap <b>Share</b><div class="inlineImg"><img src={ShareIcon} /></div>.
<p>
    <img src={ShareSessionParticipant} />
</p>
</li>
<li>Share the session link with your contacts or on social media.</li>
</ol>
<h2>Using the Enrolled Sessions Tab</h2>
<ol>
<li>On the Home page, go to the <b>Enrolled sessions</b> tab.</li>
<li>Find an enrolled session. Tap the session tile and tap <b>Share</b><div class="inlineImg"><img src={ShareIcon} /></div>.
<p>
    <img src={ShareSessionParticipant} />
</p>
</li>
<li>Share the session link with your contacts or on social media.</li>
</ol>
</TabItem>
<TabItem value="mentored portal" label="Web Portal">
<p>You can share a session link in one of the following ways:</p>
<ul>
<li>Using the <b>Home</b> tab</li>
<li>Using the <b>Enrolled sessions</b> tab</li>
</ul>
<h2>Using the Home Tab</h2>
<ol>
<li>Go to the <b>Home</b> tab.</li>
<li>Find an upcoming session and do one of the following actions: 
    <ul>
    <li>Click the session tile to view the Session Details page.</li>
    <li>Click the <b>View details</b> link to view the Session Details page.</li>
    </ul>
</li>
<li>Click <b>Share Session</b>.</li>
<li>To copy the shareable link, click <b>Copy link</b>.
<p>
    <img src={PortalShareSessionParticipant} />
</p>
</li>
<li>Share the session link with your contacts or on social media.</li>
</ol>
<h2>Using the Enrolled Sessions Tab</h2>
<ol>
<li>Go to the <b>Enrolled sessions</b> tab.</li>
<li>Find an enrolled session and do one of the following actions: 
    <ul>
    <li>Click the session tile to view the Session Details page.</li>
    <li>Click the <b>View details</b> link to view the Session Details page.</li>
    </ul>
</li>
<li>Click <b>Share session</b>.</li>
<li>To copy the link, click <b>Copy link</b>.
<p>
    <img src={PortalShareSessionParticipant} />
</p>
</li>
<li>Share the session link with your contacts or on social media.</li>
</ol>
</TabItem>
</Tabs>
