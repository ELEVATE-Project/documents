# About the Mentor Application

The <PartialExample mentored /> application is integrated with the microservices hosted by <Shiksha shiksha />. The application uses the authentication, notifications, user management, and mentoring services hosted on the <Shiksha shiksha /> Cloud. 

The following table summarizes the activities mentors and mentees can carry out using the <PartialExample mentored /> mobile application.


| Mentors | Mentees |
| ----------- | ----------- |
| Create and manage live <PartialExample mentored /> sessions.| Discover and join live <PartialExample mentored /> sessions. |
| Interact with the participants. | Interact with the mentor and other participants.|
| Collaborate with other mentors on <PartialExample mentored /> sessions. | Collaborate with other participants. |
| Use the mentor directory to get to know other mentors. | Use the mentor directory to get to know more about mentors.|
|Provide feedback on <PartialExample mentored /> sessions.| Provide feedback on <PartialExample mentored /> sessions.|
