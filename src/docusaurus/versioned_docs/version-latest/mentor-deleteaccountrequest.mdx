# Creating a Request to Delete the Account

After your request to delete the account is accepted, you will lose access to your account.

In addition to this, the following actions take place:

* Your profile is removed from the Mentor Directory.
* The sessions that you have created are deleted. 

<Admonition type="tip">
<p>You can reuse the deleted account's credentials if you wish to register again as a new <PartialExample mentored /> user in future.</p>
</Admonition>

**To create a request to delete your account, do as follows:**

<Tabs>
<TabItem value="mentored mobile application" label="Mobile App" default>
<ol>
<li>Go to the <b>Application</b> menu <div class="inlineImg"><img src={BurgerMenu}/></div> and select <b>Help</b>.</li>
<li>On the Help page, select <b>Request to delete my account</b> option from the help menu.
<p>
    <img src={MentorDeleteAccount} />
</p>
</li>
<li>Optionally, to specify the reason for deleting the account, enter the reason in the <b>Reason for deleting account</b> box.</li>
<li>Tap <b>Delete account</b>.</li>
<li>Tap <b>Yes</b> on the confirmation dialog. A request to delete the account is created.</li>
</ol>
</TabItem>
<TabItem value="mentored portal" label="Web Portal">
<ol>
<li>From the Quick Links section located in the footer of the home page, select <b>Help</b>.</li>
<li>On the Help page, select <b>Request to delete my account</b> option from the help menu.
<p>
    <img src={PortalMentorDeleteAccount} />
</p>
</li>
<li>Optionally, to specify the reason for deleting the account, enter the reason in the <b>Reason for deleting account</b> box.</li>
<li>Click <b>Delete account</b>.</li>
<li>Click <b>Yes</b> on the confirmation dialog. A request to delete the account is created.</li>
</ol>  
</TabItem>
</Tabs>
