# Using a Shared Link to View a Mentor Profile

You can view a mentor profile using a link shared by your contacts or on social media.

<Tabs>
<TabItem value="mentored mobile application" label="Mobile App" default>
<Admonition type="tip">    
<p>Users who have not registered on <PartialExample mentored /> yet, should do the following actions:</p>
<ol>
<li>Install the application.</li>
<li>Do one of the following actions:
    <ul>
    <li>Sign up to create a new account.</li>
    <li>Log in if you already have an account.</li>
    </ul>
</li>
<li>Update your profile before enrolling and joining for sessions created by the mentor.</li>
</ol>
</Admonition> 
<br />
<p>To view the mentor profile page, do as follows:</p>
<ol>
<li>Tap the shared mentor profile link. A dialog appears displaying the applications with which you can view the mentor profile. 
<br />
<br />
   <img src={OpenWithOptions} />
</li>
<li>Tap <b><PartialExample mentored /></b>. The mentor profile page appears on the mobile application.
<br />
<br />
    <img src={MentorProfilePage} />
</li>  
</ol>
</TabItem>
<TabItem value="mentored portal" label="Web Portal">
<Admonition type="tip"> 
<p>Users who have not registered on <PartialExample mentored /> yet, should do the following actions:</p>
<ol>
<li>Open the web portal.</li>
<li>Do one of the following actions:
    <ul>
    <li>Sign up to create a new account.</li>
    <li>Log in if you already have an account.</li>
    </ul>
</li>
<li>Update your profile before enrolling and joining for sessions created by the mentor.</li>
</ol>
</Admonition> 
<br />
<p>To view the mentor profile page, click the shared mentor profile link. The mentor profile page appears on the web portal.</p>
<br />

</TabItem>
</Tabs>
