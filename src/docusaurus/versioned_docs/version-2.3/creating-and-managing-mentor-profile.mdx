# Creating and Managing a Mentor Profile

On the Profile page, you can do the following actions:

<ul>
<li>Update your profile details to help mentees get to know you.</li>
<li>View the number of sessions conducted.</li>
<li>View your session ratings.</li>
</ul>

<Tabs>

## Updating your Profile

<Admonition type="note">
<p>You can use the application only after updating your profile.</p>
</Admonition>

<b>To update your profile, do as follows:</b>

<Tabs>
<TabItem value="mentored mobile application" label="Mobile App" default>
<ol>
<li>On the Profile page, tap <b>Edit profile</b>.
<br />
<br />
    <img src={EditMentorProfile} />
</li>
<li>On the Profile details page, enter your details.
<br />
<br />
    <img src={CreateProfile} />
</li>
<li>If you want to add a profile picture, tap <b>Add image</b>. Select <b>Take Photo</b> or <b>Choose from Library</b>.</li>
<li>Tap <b>Submit</b>. The profile details are updated.</li>
</ol>
</TabItem>
<TabItem value="mentored portal" label="Web Portal">
<ol>
<li>On the Profile page, click <b>Edit profile</b>.
<br />
<br />
    <img src={PortalEditMentorProfile} />
</li>
<li>On the Profile details page, enter your details.
<br />
<br />
    <img src={PortalCreateProfile} />
</li>
<li>If you want to add a profile picture, click <b>Add image</b> and select the image.</li>
<li>Click <b>Submit</b>. The profile details are updated.</li>
</ol>
</TabItem>
</Tabs>
